# Prisma MongoDB Backend Project

This project demonstrates a Node.js backend using **Prisma ORM** with **MongoDB**. It includes user authentication, cookie handling, error handling, and token management.

## 📦 Tech Stack

- Node.js
- Express.js
- Prisma ORM
- MongoDB (Atlas)
- JSON Web Token (JWT)
- bcryptjs
- cookie-parser
- dotenv
